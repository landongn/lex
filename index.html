    lex parsimoniae - A simple javascript microframework, but no simpler.

    Project maintained by landongn Hosted on GitHub Pages — Theme by mattgraham

    Howdy.

    Lex is a javascript microframework. You could almost call it a pattern, more that a framework.

    It allows for a very terse, simple namespace abstraction, and some minor conveniences (bundled with EventEmitter, Mustache Templates, some functional goodies) that allow you to iterate fast, loosely coupled and elegant javascript apps.

    Included is a single sample that shows you how to get started. Try it out in the ./samples/basic_modules/ directory.
